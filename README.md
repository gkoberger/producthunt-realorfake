Don't use this yet. It isn't ready. But if you must:

SETUP
-----

    git clone https://github.com/gkoberger/big-gulp.git
    cd big-gulp
    npm install
    bower install

    # You probably want to make this your own
    rm -rf .git
    git init

USING
-----

    gulp serve

Go to http://localhost:8080
